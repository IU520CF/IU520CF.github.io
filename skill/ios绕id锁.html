# ios 绕过 id 锁

# checkra1n

官网：https://checkra.in/

优点：最起码解决手机平板不是砖

缺点：不能插卡打电话，变越狱机了

处理器：A9 A10

ios系统: 12-14

(外国大神，团队主角预计发布最后一版就不再维护了，最后一版低端处理器适配到15版本，目前没有说谁接盘【开源免费软件，也只有不为生活发愁，又热爱这个技术的人才能接盘了。最主要的是牛逼的技术】)

环境：linux 和 mac

安装 checkra1n

按官网的来吧，mac下载正常安装，linux系统Debian/ubuntu 官网有命令安装，其他版本源码安装

注意：苹果手机需原装数据线。

刷机有风险：正常手机就算了，本身等于砖可以试下【变砖自己负责，与本教程无关，本教程不推荐刷机】

提示：界面全英文（推荐有道词典截屏翻译，多试几次）

iphone 优点：软硬件都是自己的技术解耦性好，同配置性能甩开安卓两倍多，后台机制好，不是摔坏最少用四年

iphone 缺点：

​ 1、ios版本能升到机器支持的最大版本，不能降级，高版本对新手机更好，低端处理器，版本太高会卡顿、发热、掉电快。只要流程别升级。

​ 2、买二手，或者毛毛躁躁的人【把发票邮箱弄丢后】手机卡换了，邮箱长时间用被回收废了，密码输入多次被锁，官方不给解决，只能越狱，越狱后安全性很低，所有软件都可以最高权限运行。安装不好的软件，软件商想控制你手机，变肉鸡分分秒秒的事情【所以越狱后还是安装正版来源的软件吧】。

本人以后将不再购买任何苹果设备

之前官网买了ipad21版，京东买了macbook 。为了生态买了有锁的iphone，查了发现越狱暂时不支持ios15.1一天资料心累了，以后不会再买苹果设备了。

错误密码输入过多次会被停用,优点害怕，不下载软件还是退出登录吧，我怕被封。。。

以后手机买可以随意刷机的

台式自己配置

平板【实际没什么用，之前买是看电子书，后来发现还是伤眼睛，需要的时候买纸质书,电脑像素高点也一样。】

想法：其实我们用的生态，我理解的生态，

1、手机文件和电脑文件有一个公用区域。

2、代码跑服务器，电脑编辑器只负责写入，运算都跑服务器 （JetBrains全家桶都帮你实现了，如果工作了，那就买正版，它是剑客手中的宝剑, 枪手心中的枪械。我们大多都是学生时代过来的，学生免费，让你用。而且服务态度特别好。值得这个价格。作为一个开发人员，我们自己不正视版权，别人会正视吗？以后大部分人都会编程，好的东西都不想付出钱，开发的动力都没了）

【家用台式机配个强点的, 假设数十个，内网 vps 服务器。各种开发环境搭建。】---《未来规划》

3、密码管理器：纸制记事本重要点的密码。谷歌浏览器【平常网站密码，不是很重要的，不会对财产造成损失的，谷歌存储。】 KeePassX 电脑密码本。本地加密存储，不会上传网络。不能同步，安全性应该还行我在用。

4、图床：写博客什么的图片存放方便。csdn 对外链图片不友好【我也是有点头疼要不要用，已经搭建好了。】

5、网盘：和第一个文件同步，一个东西其实，这个可以合并，多端同步本来就会用的服务器，服务器，也存储一份，方便分享文件【太占服务器空间这个事情，其实可以在多端同步区域再开辟一块空间，一般存放可能分享的文件，这样也就解决了大量占用服务器空间的问题】
